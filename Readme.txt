Matlab demo code for "Single Image Super-Resolution With Learning Iteratively Accurate Non-Linear Mapping Between Low- And High-Resolution Sparse Representations" submitted to neurocomputing by Kun Zeng (zengkun@xmu.edu.cn)


This code is for academic purpose only. Not for commercial/industrial activities.


***********************************************************************************************************
***********************************************************************************************************

Usage:

demo_SR.m - demonstrate super-resolution using S3cSR.m

function:

S3cSR.m - realize super resolution given the model parameters

Folders:

Set5 and Set14 - test images 

Model - "x2.mat" "x3.mat" and "x4.mat" are model parameters used for upscaling factors 2,3 and 4 seperately 

### Requirements and Dependencies
- MATLAB (we test with MATLAB R2014a on Windows 10)
- [SPAMS](http://spams-devel.gforge.inria.fr/index.html)
